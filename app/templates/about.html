About Company Management System

About This System

The Company Management System is a comprehensive web-based application designed to streamline human resources and employee management operations for organizations of all sizes.

Key Features:
  • Employee Management: Add, edit, view, and delete employee records with complete information
  • Attendance Tracking: Mark and monitor employee attendance with multiple status options
  • Salary Management: Generate payroll, track salary payments, and manage allowances/deductions
  • Search & Filter: Quickly find employees by name, email, department, or status
  • Data Export: Export employee records to CSV format for external analysis
  • Dashboard Analytics: View company statistics and departmental distribution
Technology Stack
Backend:
  • Python 3.9+ - Programming Language
  • Flask - Web Framework
  • SQLAlchemy - ORM for Database
  • NeonDB - Database
Frontend:
  • Bootstrap 5 - UI Framework
  • HTML5 - Markup
  • CSS3 - Styling
  • JavaScript - Interactivity
System Requirements
  • Python 3.9 or higher
  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Internet connection (for CDN resources)
  • Minimal server resources (runs on free hosting platforms)
